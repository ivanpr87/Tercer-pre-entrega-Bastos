Acerca del Proyecto

Bienvenido al proyecto de gestión de autos y clientes. Este proyecto tiene como objetivo facilitar el manejo de información relacionada con autos y clientes, brindando una plataforma intuitiva y eficiente.

En esta aplicación, podrás realizar diversas acciones, como agregar nuevos autos, visualizar la lista de autos registrados, llevar un registro de las actividades realizadas, añadir clientes y gestionar su información, entre otras funciones útiles.

La interfaz amigable y las funcionalidades bien organizadas hacen que este proyecto sea ideal para aquellos que buscan una solución integral para la gestión de datos en el ámbito automotriz.
